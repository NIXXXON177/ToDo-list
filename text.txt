1. main.js:
- Основной файл инициализации приложения
- Создает основную структуру приложения
- Инициализирует модальные окна (авторизации и задач)
- Запускает авторизацию
- Инициализирует обработчики событий
- Инициализирует тему приложения

2. auth.js:
- Управляет авторизацией пользователей
- Обрабатывает вход пользователей
- Обрабатывает регистрацию новых пользователей
- Хранит данные пользователей в localStorage
- Переключает формы входа/регистрации
- Валидирует данные пользователя

3. createElements.js:
- Создает все HTML элементы приложения
- Формирует основную структуру приложения
- Создает модальные окна авторизации
- Создает модальные окна для работы с задачами
- Настраивает атрибуты и классы элементов

4. domHandler.js:
- Управляет взаимодействием с DOM
- Обрабатывает действия с задачами (добавление, удаление, редактирование)
- Отрисовывает список задач
- Управляет фильтрацией задач
- Создает и управляет модальными окнами подтверждения
- Обрабатывает события клика и клавиатуры

5. events.js:
- Инициализирует обработчики событий
- Управляет отображением информации о пользователе
- Обрабатывает выход из системы
- Показывает информацию о пользователях
- Создает модальные окна подтверждения для выхода

6. taskManager.js:
- Управляет задачами пользователя
- Добавляет новые задачи
- Удаляет задачи
- Обновляет существующие задачи
- Отмечает задачи как выполненные
- Сохраняет задачи в localStorage

7. theme.js:
- Управляет темой приложения
- Переключает между светлой и темной темой
- Сохраняет выбранную тему в localStorage
- Применяет тему при загрузке приложения

8. utils.js:
- Содержит вспомогательные функции
- Создает информационные модальные окна
- Отображает сообщения пользователю
- Предоставляет общие утилиты для других модулей
